# my-develop-plugin

特定のプロジェクトに依存しない開発用スキルをまとめた Claude Code プラグインです。

## 使い方

```bash
/skill <skill-name> <task>
```

## スキル

`skills/` 配下に開発向けのスキルを収録しています。

### owasp-security-review

OWASP Cheat Sheet Series に基づくセキュリティレビューと実装支援を行います。

## 開発

ローカルでの構成検証:

```bash
claude plugin validate .
```
